JavaScript desde un archivo externo

Este es un pequeño párrafo

Los archivos externos son muy prácticos cunado un mismo código se emplea en distintos archivos.

Manejando JavaScript en archivos externos tenemos también las siguientes ventajas:

  • Separas el lenguaje HTML del código JavaScript
  • Se hace más fácil el mantenimiendo tanto del lenguaje HTML y del código de JavaScript
  • Los archivos JavaScript en caché acelararán la carga de tus páginas
